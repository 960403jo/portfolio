## 사용 목적

이 Notion 페이지는 포트폴리오 웹페이지의 원본 콘텐츠 기준이다.

## Notion 내용 반영 규칙

```text
- Notion에 있는 프로젝트명, 기간, 역할, 기술스택을 우선한다.
- Notion에 있는 문장은 웹페이지용 문장으로 다듬을 수 있다.
- 사실관계는 변경하지 않는다.
- Notion에 없는 수치 성과는 임의로 만들지 않는다.
- 불명확한 정보는 docs/open-questions.md에 기록한다.
```

## Notion Export 방법

```text
1. Notion 페이지 우측 상단 ... 클릭
2. Export 선택
3. Markdown & CSV 선택
4. Export
5. 생성된 Markdown 내용을 docs/notion-source.md에 복사
```
