# Installation Guide

## 1. Next.js 설치는 직접 수행

```bash
npx create-next-app@latest portfolio
cd portfolio
```

권장 옵션:

```text
TypeScript: Yes
ESLint: Yes
Tailwind CSS: Yes
App Router: Yes
src directory: Yes
Import alias: @/*
```

## 2. 이 압축 파일 해제

프로젝트 루트에 압축을 해제한다.

```text
portfolio/
├─ AGENTS.md
├─ docs/
├─ supabase/
├─ public/
├─ scripts/
└─ .github/
```

## 3. 스크립트 권한

```bash
chmod +x scripts/*.sh
```
