# Agent Rules

## 기준 문서 우선순위

```text
1. AGENTS.md
2. docs/agent-status.md
3. docs/agent-handoff.md
4. docs/agent-messages.md
5. docs/notion-source.md
6. docs/notion-content-extraction.md
7. docs/portfolio-content.md
8. docs/portfolio-requirements.md
9. docs/site-map.md
10. docs/page-spec.md
11. docs/component-spec.md
12. docs/design-system.md
13. docs/next-app-design.md
```

## 공통 규칙

```text
- Notion 원본을 최우선 기준으로 한다.
- Notion에 없는 내용은 임의로 만들지 않는다.
- 링크가 미확정이면 docs/open-questions.md에 기록한다.
- 진행상황은 docs/agent-status.md에 기록한다.
- Agent 간 전달사항은 docs/agent-handoff.md에 기록한다.
- Agent 간 대화형 기록은 docs/agent-messages.md에 기록한다.
- 포트폴리오 내용은 과장하지 않는다.
- SEO metadata를 작성한다.
- 모바일 반응형을 기본으로 한다.
```
