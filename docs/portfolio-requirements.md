# Portfolio Requirements

## 1. 목표

Notion에 정리된 경력, 프로젝트, 기술 스택, 개발 방식을 기반으로 Next.js + Supabase 포트폴리오 웹페이지를 구축한다.

## 2. 콘텐츠 기준

```text
Primary Source:
- docs/notion-source.md

Parsed Source:
- docs/notion-content-extraction.md

Display Source:
- docs/portfolio-content.md
```

## 3. 기능 요구사항

```text
- Hero
- About
- Skills
- Projects
- Experience
- AI Workflow
- Contact
- 반응형 레이아웃
- SEO metadata
- Open Graph metadata
- Contact form 선택
```
